# Result Directory

このディレクトリには、モデル学習の実行結果が保存されます。

## ディレクトリ構成

### YYYYMMDDhhmmss/
学習実行時に自動生成されるタイムスタンプ形式のディレクトリです。
- 形式：`年月日時分秒` (例: `20260127001607`)
- 各学習セッションの結果が個別に保存されます

各ディレクトリには以下のファイルが含まれます：
- `best_model.pth` - 学習中に最良の性能を示したモデルの重み
- `config.json` - 学習時のハイパーパラメータ設定
- その他のログファイルや評価結果

## 使用方法

学習を実行すると、自動的に新しいタイムスタンプディレクトリが作成され、結果が保存されます。

```python
# 学習実行の例
python train.py
```

学習結果をロードする場合：

```python
import torch
import json

# 設定ファイルの読み込み
with open('result/20260127001607/config.json', 'r') as f:
    config = json.load(f)

# モデルのロード
model.load_state_dict(torch.load('result/20260127001607/best_model.pth'))
```

## 注意事項

- タイムスタンプディレクトリとその内容はGitの管理対象外です
- 学習結果ファイルは容量が大きいため、リポジトリには含めません
- 各実行結果は履歴として保持されるため、古い結果は手動で削除してください
- このREADME.mdファイルのみがリポジトリに含まれます

## ディレクトリ管理

不要な学習結果を削除する場合：

```bash
# 特定の結果を削除
rm -rf result/20260127001607

# 古い結果をまとめて削除（最新5件を残す例）
ls -t result/ | tail -n +6 | xargs -I {} rm -rf result/{}
```
