# Data Directory

このディレクトリには、株価データと学習用データが保存されます。

## ディレクトリ構成

### csv_YYYYMMDD_hhmmss/
株価データの取得・処理時に自動生成されるディレクトリです。
- タイムスタンプ形式：`csv_年月日_時分秒`
- 各銘柄の株価データがCSV形式で保存されます
- 学習データの前処理結果も含まれます

### Excelファイル
- `jpx400_data.xlsm` - JPX400銘柄のマスターデータ
- `jpx400_market_speed_2.xlsx` - マーケット速報データ

## データ取得方法

データ取得スクリプトを実行すると、自動的に新しい`csv_YYYYMMDD_hhmmss`ディレクトリが作成され、データが保存されます。

```python
# データ取得の例
python scripts/fetch_stock_data.py
```

## 注意事項

- `csv_*` ディレクトリはGitの管理対象外です（容量が大きいため）
- CSVファイル、Excelファイルもリポジトリには含まれません
- データは各自の環境で取得・生成してください
- このREADME.mdファイルのみがリポジトリに含まれます

## データ形式

各CSVファイルには以下のような株価情報が含まれます：
- 日付
- 始値、高値、安値、終値
- 出来高
- テクニカル指標（移動平均など）
