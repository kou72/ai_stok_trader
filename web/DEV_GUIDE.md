# 開発ガイド

ビルド不要で開発できる方法を説明します。

## 開発モード（推奨）

### 1. 必要なパッケージのインストール（初回のみ）

```bash
cd web/frontend
npm install
```

### 2. 開発サーバーの起動

#### 方法1: 1コマンドで両方起動（簡単！推奨）

```bash
cd web/frontend
npm run dev:all
```

これで Flask（ポート5000）と Vite（ポート5173）が同時に起動します。

#### 方法2: 別々のターミナルで起動

**ターミナル1: Flaskサーバー**
```bash
# プロジェクトルートから
python web/app.py
```

**ターミナル2: Viteフロントエンド**
```bash
cd web/frontend
npm run dev
```

### 3. ブラウザでアクセス

```
http://localhost:5173
```

### 開発モードの利点

- ファイルを保存すると自動的にブラウザがリロードされる
- ビルド不要で即座に変更が反映される
- エラーがブラウザに表示される
- ホットモジュールリプレースメント（HMR）で高速

## 本番用ビルド

本番環境やFlaskサーバーのみで動かす場合は、ビルドが必要です。

```bash
cd web/frontend
npm run build
```

その後、Flaskサーバーのみを起動：

```bash
python web/app.py
```

ブラウザで http://localhost:5000 にアクセス

## トラブルシューティング

### ポート5173が既に使用されている

別のアプリケーションがポート5173を使用している場合、Viteが自動的に次の空きポート（5174など）を使用します。ターミナルに表示されるURLを確認してください。

### APIリクエストが失敗する

Flaskサーバー（ポート5000）が起動しているか確認してください。

### 変更が反映されない

1. ブラウザのキャッシュをクリア（Ctrl+Shift+R / Cmd+Shift+R）
2. Vite開発サーバーを再起動

## ファイル編集時のワークフロー

1. VSCodeなどでファイルを編集
2. 保存（Ctrl+S / Cmd+S）
3. ブラウザが自動的にリロード
4. 変更を確認

ビルドコマンドは不要です！
